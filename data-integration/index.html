Distribute data to your customers

Serving analytical workloads generated by thousands of customer from a single warehouse is not feasible. GoodData organizes analytical data of your customers into workspaces. This enables per-customer customizability and access control while you can manage versions and releases using our lifecycle management tooling.

Declarative distribution service efficiently pushes deltas from your retrieved data into thousands of customer workspaces.

Need more data preparation?

  • The Data Integrator can run initial de-duplication, enrichment with computed fields, and manage how downloaded data is merged into persistent tables.
  • The SQL Executor can orchestrate a flow of SQL transformations to pre-aggregate, de-normalize, process time-series data, build snapshots, consolidate multiple data sources and much more.
  • You can deploy and schedule custom scripts to (re-)train machine learning models or to perform any other custom data preparation logic.
--Transform opportunity fields history
SELECT r.*, o.value FROM (
    SELECT MIN(d._VALID_FROM) AS _VALID_FROM, r.record_id, r.field FROM
    (SELECT DISTINCT record_id, field FROM opportunity_fields_history) r
    INNER JOIN (SELECT DISTINCT _VALID_FROM, record_id FROM opportunity_fields_history) d
        ON d.record_id = r.record_id

Schedule and monitor

Schedule the data ingestion task or invoke it by an API call or from the administration user interface. Set up the dependencies between tasks. Get alerts about failures by e-mail or integrate with services such as PagerDuty. See the tasks execution history, access log files with details about possible failures.
